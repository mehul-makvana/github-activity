## ðŸš€ Installation

### 1. Clone the Repository

```bash
git clone <repository-url>
cd SportsBettingApp
```

### 2. Install Dependencies

The project uses a root-level script to install dependencies for both frontend and backend:

```bash
# Install backend dependencies
cd betting-server
npm install

# Install frontend dependencies
cd ../client
npm install
```

### 3. Running the Application on Your Local

From the root directory:

```bash
npm run dev
```
